## 第 1 关：旭日（`firstMandate`，1661–1675）

| 项目 | 内容 |
| --- | --- |
| 回合 | 15 年（每回合 1 年） |
| 开局资源 | 财政 2 / 权力 2 / 合法性 2 |
| 胜利条件 | 财政 ≥ 6，权力 ≥ 6，合法性 ≥ 5 |

### 一、机制列表（按“发生顺序/优先级”）

| # | 机制 | 要点 |
| --- | --- | --- |
| 1 | 年初资源与抽牌机制 | 先执行年初状态效果，再把 `经费 += 财政`。抽牌次数由：权力阈值基数 + 上回合延迟修正 + 多年延迟修正 + 状态修正 +（若有）反法同盟掷骰修正组成，最少抽 1。 |
| 2 | 事件填充机制（A-C 常规槽） | 第 1 年固定前缀事件：`殖民贸易机遇` + `官僚拖延`。之后按加权序列填充（去重抽取，优先补空槽）。 |
| 3 | 年末未处理事件机制 | 有害事件在年末按槽位顺序结算惩罚。`地方总督坐大`是特殊升级调度事件：本年未处理不立即罚值，而是给该槽挂“次年升级”标记。 |
| 4 | 槽位升级机制（Power Vacuum → Major Crisis） | 下一个事件阶段开始时，若槽位仍是`地方总督坐大`且已挂标记，自动替换为`王权危机`。 |
| 5 | 脚本历史战事：遗产战争（1667–1669） | 可选“进攻”（脚本攻击）：花 2 经费，权力 +1，且有 50% 概率财政 +1。一旦进攻，进入“反法同盟压力”：自进攻当回合起 **5 年内**，每年以 40% 概率令抽牌尝试 -1（最终仍至少 1 抽）。 |
| 6 | 第一关通胀触发机制 | 当 `财政 + 权力 + 合法性 >= 12` 时激活通胀。激活后，仅带 `通胀` 标签卡牌在“弃牌回洗入牌库”时按实例叠加费用。 |
| 7 | 有限耐久牌耗尽后果 | `王室征收`/`王室干预`有“剩余次数”机制；耗尽后会从循环中移除并触发惩罚：征收耗尽财政 -1；干预耗尽权力 -1。 |

##### 王室干预 · `crackdown`

**完整 Prompt：**

```text
请生成横向图片（横版构图）。为“王室干预”（`crackdown`）创作卡牌插画。风格改为 17 世纪法兰西历史题材卡通插画：清晰粗线轮廓、人物与服饰比例轻微夸张、平涂主导并辅以柔和体积阴影；配色保持深蓝灰天光、暖灰石材、暗金点缀，非照片写实。

核心叙事必须清晰：**来自国王的命令，正在明确指向一群人**。画面中上方设置一只佩戴王室袖饰的手臂（可只露前臂与手），从帷幕或拱门阴影中伸出，手持加蜡封的王令卷轴；卷轴末端或手势方向明确朝向下方聚集人群。

构图：上方为拱门/宫廷走廊结构与“下达命令的手 + 王令卷轴”；下方为被指向的人群（约 8～15 人）与少量执行者剪影（可含 1～2 名火枪手），人群姿态体现被点名后的紧张、回头、后退、低语等反应。人物面部不可辨识，不画具体五官。

光影与情绪：命令源头区域（手与卷轴）受较亮冷光，人群区域偏暗且密集，形成“上方命令、下方受令”的压迫对比。整体气质是肃杀、果断、王权直接干预（不是屠杀场面，不是狂欢节）。

隐喻目标：“国王命令直接指向特定人群”用于表达以强制手段精准压制一个有害危机；卷轴和服饰可有轻微磨损，暗示干预手段可用但会消耗政治信用。

硬性限制：不得出现现代物件、现代警械或枪械细节、血腥元素、可识别真实历史人物肖像、单一著名地标特写；画面任何位置都不得出现清晰可读文字（包括可辨认的拉丁文/法文/英文、旗帜标语、铭文、签名）。
```

### 五、叙事参考（导语 / 文案基底）

西班牙哈布斯堡末王卡洛斯二世无嗣，帝国遗产危及欧洲均势；波旁以安茹公爵腓力（费利佩五世）系血缘与遗嘱自辩，哈布斯堡以奥地利大公查理系延续自辩；战争升级为大同盟式多国冲突。1713《乌得勒支和约》等确立费利佩可留西班牙王位但法西不得合并，条约均势取代单一普世王朝帝国想象。路易十四留下的强国家机器与欧洲遏制记忆，在本关可作为“高能力 / 高外部反弹”的双刃叙事。
