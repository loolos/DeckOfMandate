> 口径：以当前代码实现为准，覆盖 `firstMandate` → `secondMandate` 的真实运行逻辑。  
> 主要依据：`src/data/*`、`src/logic/*`、`src/app/gameReducer.ts`、`src/locales/*`。

## 第 1 关：旭日（`firstMandate`，1661–1675）

- 回合：15 年（每回合 1 年）
- 开局资源：财政 2 / 权力 2 / 合法性 2
- 胜利条件：财政 ≥ 6，权力 ≥ 6，合法性 ≥ 5

### 一、机制列表（按“发生顺序/优先级”）

1. **年初资源与抽牌机制**
   - 先执行年初状态效果，再把 `经费 += 财政`。
   - 抽牌次数由：权力阈值基数 + 上回合延迟修正 + 多年延迟修正 + 状态修正 +（若有）反法同盟掷骰修正组成，最少抽 1。

2. **事件填充机制（A-C 常规槽）**
   - 第 1 年固定前缀事件：`殖民贸易机遇` + `官僚拖延`。
   - 之后按加权序列填充（去重抽取，优先补空槽）。

3. **年末未处理事件机制**
   - 有害事件在年末按槽位顺序结算惩罚。
   - `地方总督坐大`是特殊升级调度事件：本年未处理不立即罚值，而是给该槽挂“次年升级”标记。

4. **槽位升级机制（Power Vacuum → Major Crisis）**
   - 下一个事件阶段开始时，若槽位仍是`地方总督坐大`且已挂标记，自动替换为`王权危机`。

5. **脚本历史战事：遗产战争（1667–1669）**
   - 可选“进攻”（脚本攻击）：花 2 经费，权力 +1，且有 50% 概率财政 +1。
   - 一旦进攻，进入“反法同盟压力”：之后若干年每年以 40% 概率令抽牌尝试 -1（最终仍至少 1 抽）。

6. **第一关通胀触发机制**
   - 当 `财政 + 权力 + 合法性 >= 12` 时激活通胀。
   - 激活后，仅带 `通胀` 标签卡牌在“弃牌回洗入牌库”时按实例叠加费用。

7. **有限耐久牌耗尽后果**
   - `王室征收`/`王室干预`有“剩余次数”机制；耗尽后会从循环中移除并触发惩罚：
     - 征收耗尽：财政 -1
     - 干预耗尽：权力 -1

### 二、事件列表（费用 / 标签 / 处理与不处理作用 / 历史背景 / 发生机制）

> 说明：标签含义为 `有害/机遇/史实/持续`。`史实`仅叙事标识，不自带额外数值效果。

#### 1) 常规随机池（第一关）

1. **宫廷超支 `budgetStrain`**
   - 费用：2（支付经费处理）
   - 标签：有害
   - 处理：支付 2 经费解决
   - 不处理：财政 -1
   - 历史背景：凡尔赛扩建与宫廷礼仪推升常年支出
   - 发生机制：第一关常规池，权重 3

2. **巴黎骚动 `publicUnrest`**
   - 费用：仅可用干预处理（不能用经费直付）
   - 标签：有害
   - 处理：用`王室干预`（或功能等效牌）清除
   - 不处理：合法性 -1
   - 历史背景：粮价与流言引发首都骚动
   - 发生机制：第一关常规池，权重 3

3. **官僚拖延 `administrativeDelay`**
   - 费用：1
   - 标签：有害
   - 处理：支付 1 经费
   - 不处理：下回合抽牌修正 -1
   - 历史背景：地方官延缓执行中央政令
   - 发生机制：第一关常规池，权重 2；第 1 年固定开场前缀之一

4. **殖民贸易机遇 `tradeOpportunity`**
   - 费用：1
   - 标签：机遇
   - 处理：支付 1，经费结算后财政 +1
   - 不处理：无惩罚
   - 历史背景：海外航线与商人集团寻求王室支持
   - 发生机制：第一关常规池，权重 2；第 1 年固定开场前缀之一

5. **贵族阻挠 `politicalGridlock`**
   - 费用：2
   - 标签：有害
   - 处理：支付 2 经费
   - 不处理：获得`权力流失`3 回合（每回合抽牌尝试 -1）
   - 历史背景：大贵族联合牵制新税与政令推进
   - 发生机制：第一关常规池，权重 2

6. **地方总督坐大 `powerVacuum`**
   - 费用：2 或干预
   - 标签：有害 + 持续
   - 处理：支付 2 或用干预
   - 不处理：年末登记“升级为王权危机”（非立即数值惩罚）
   - 历史背景：地方权力中心绕开宫廷形成事实自治
   - 发生机制：第一关常规池，权重 1；属于升级调度源头

7. **王权危机 `majorCrisis`**
   - 费用：仅干预
   - 标签：有害 + 持续
   - 处理：必须干预
   - 不处理：每年末合法性 -1 且下回合抽牌修正 -1，并持续留场
   - 历史背景：中央权威受挑战后演变为结构性治理危机
   - 发生机制：仅由`地方总督坐大`未处理后在下一事件阶段转化产生

#### 2) 脚本历史事件（第一关）

8. **遗产战争 `warOfDevolution`**
   - 费用：脚本“进攻”消耗 2（非普通付费解事件）
   - 标签：史实
   - 处理：可发动进攻（权力 +1；50% 概率财政 +1）
   - 不处理：无直接惩罚，但会占位到 1669 窗口结束
   - 历史背景：1667–1668 年法兰西围绕西属尼德兰/弗朗什孔泰发动战争，促发欧洲制衡
   - 发生机制：仅在 1667–1669 年脚本注入

## 第 2 关：荣光之重（`secondMandate`，1676–1700）

- 回合：25 年（每回合 1 年）
- 开局资源：
  - 关卡定义值：财政 3 / 权力 3 / 合法性 3
  - 实际直开（standalone 草案默认）：财政 7 / 权力 7 / 合法性 5
  - 承接模式（continuity）：继承第一关结算资源（并按分支追加修正）
- 胜利条件（以运行时代码为准）：
  - 当前年需 ≥ 1696
  - `欧洲警觉`必须已解除（通常通过处理`里斯维克和约`）

### 一、机制列表（按“发生顺序/优先级”）

1. **欧洲警觉（开章+年度）**
   - 开章即时权力损失：当前版本为 **0**（已取消开章扣权机制）。
   - 年度基础漂移（回合开始）：
     - 设 `x = 财政 + 权力 + 合法性`，`y = 欧洲警觉进度`，`k = x - 12 - y*3`。
     - 若 `k > 0`：有 `k*10%` 概率进度 `+1`。
     - 若 `k < 0`：有 `(-k)*10%` 概率进度 `-1`。
     - 概率按 100% 封顶，进度始终夹在 `1~10`。
   - 年度外压注入（事件阶段）：
     - 进度 `1~5`：以 `进度*20%` 概率额外注入 1 个欧洲警觉事件。
     - 进度 `6~10`：保底注入 1 个，再以 `(进度-5)*20%` 概率注入第 2 个。
   - 阶段文本（状态栏背景）：
     - `1~2` 缓和，`3~4` 警觉，`5~6` 遏制，`7~8` 敌视，`9~10` 冲突。

2. **脚本历史线注入窗口**
   - 按年份窗口注入：`扩张余震/谨慎王权` → `奈梅亨和约` → `撤销南特敕令` → `奥格斯堡同盟` → `九年战争` → `里斯维克和约`。
   - 注：`地方战事`不属于脚本注入，而是欧洲警觉补充池中的额外外压事件。

3. **反法情绪阈值机制**
   - 当 `权力 + 财政 > 20` 时激活`反法情绪`状态。
   - 当总和大于 20 时，所有“支付经费解决”的事件费用按超额分段上调：每满 `+5`，费用 `+1`。
   - 当该状态存在时，每回合结束向抽牌堆加入 1 张`反法遏制`（费用 2；抽到时合法性 -1；打出后直接消失，不进入弃牌堆）。
   - 当降回阈值内时自动移除该状态并取消上述费用加成。

4. **宗教宽容的副作用触发**
   - 存在`宗教宽容`时，每年 30% 概率额外生成`宗教矛盾`（若场上尚无该事件）。

5. **胡格诺镇压链机制**
   - 选择“严厉镇压”后：添加`遏制胡格诺-剩余势力(3)`并向牌库加入 3 张`镇压胡格诺派`。
   - 每打出 1 张，剩余势力 -1；归零时移除该状态并清除所有该临时牌。
