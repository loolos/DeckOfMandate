> 口径：三关均以当前代码实现为准。太阳王关卡定义、模板与注册见 `src/levels/sunking/`（各章 `chapters/*.ts`、`chapter2Transition.ts`、`chapter3Transition.ts`、`templates/`、`locales/`）；关卡注册在 `src/levels/registerAll.ts`；通用逻辑见 `src/logic/*`，**状态归一**见 `src/logic/normalizeGameState.ts`，**胜负**见 `src/logic/turnFlow.ts` 中 `evaluateVictory` / `evaluateTimeDefeat`（第三关 `successionWar` 对 `evaluateTimeDefeat` 无回合判负）。

## 第 1 关：旭日（`firstMandate`，1661–1675）

- 回合：15 年（每回合 1 年）
- 开局资源：财政 2 / 权力 2 / 合法性 2
- 胜利条件：财政 ≥ 6，权力 ≥ 6，合法性 ≥ 5

### 一、机制列表（按“发生顺序/优先级”）

1. **年初资源与抽牌机制**
  - 先执行年初状态效果，再把 `经费 += 财政`。
  - 抽牌次数由：权力阈值基数 + 上回合延迟修正 + 多年延迟修正 + 状态修正 +（若有）反法同盟掷骰修正组成，最少抽 1。
2. **事件填充机制（A-C 常规槽）**
  - 第 1 年固定前缀事件：`殖民贸易机遇` + `官僚拖延`。
  - 之后按加权序列填充（去重抽取，优先补空槽）。
3. **年末未处理事件机制**
  - 有害事件在年末按槽位顺序结算惩罚。
  - `地方总督坐大`是特殊升级调度事件：本年未处理不立即罚值，而是给该槽挂“次年升级”标记。
4. **槽位升级机制（Power Vacuum → Major Crisis）**
  - 下一个事件阶段开始时，若槽位仍是`地方总督坐大`且已挂标记，自动替换为`王权危机`。
5. **脚本历史战事：遗产战争（1667–1669）**
  - 可选“进攻”（脚本攻击）：花 2 经费，权力 +1，且有 50% 概率财政 +1。
  - 一旦进攻，进入“反法同盟压力”：自进攻当回合起 **5 年内**，每年以 40% 概率令抽牌尝试 -1（最终仍至少 1 抽）。
6. **第一关通胀触发机制**
  - 当 `财政 + 权力 + 合法性 >= 12` 时激活通胀。
  - 激活后，仅带 `通胀` 标签卡牌在“弃牌回洗入牌库”时按实例叠加费用。
7. **有限耐久牌耗尽后果**
  - `王室征收`/`王室干预`有“剩余次数”机制；耗尽后会从循环中移除并触发惩罚：
    - 征收耗尽：财政 -1
    - 干预耗尽：权力 -1

### 二、事件列表（费用 / 标签 / 处理与不处理作用 / 历史背景 / 发生机制）

> 说明：标签含义为 `有害/机遇/史实/持续`。`史实`仅叙事标识，不自带额外数值效果。

#### 1) 常规随机池（第一关）

1. **宫廷超支 `budgetStrain`**
  - 费用：2（支付经费处理）
  - 标签：有害
  - 处理：支付 2 经费解决
  - 不处理：财政 -1
  - 历史背景：凡尔赛扩建与宫廷礼仪推升常年支出
  - 发生机制：第一关常规池，权重 3
2. **巴黎骚动 `publicUnrest`**
  - 费用：仅可用干预处理（不能用经费直付）
  - 标签：有害
  - 处理：用`王室干预`（或功能等效牌）清除
  - 不处理：合法性 -1
  - 历史背景：粮价与流言引发首都骚动
  - 发生机制：第一关常规池，权重 3
3. **官僚拖延 `administrativeDelay`**
  - 费用：1
  - 标签：有害
  - 处理：支付 1 经费
  - 不处理：下回合抽牌修正 -1
  - 历史背景：地方官延缓执行中央政令
  - 发生机制：第一关常规池，权重 2；第 1 年固定开场前缀之一
4. **殖民贸易机遇 `tradeOpportunity`**
  - 费用：1
  - 标签：机遇
  - 处理：支付 1，经费结算后财政 +1
  - 不处理：无惩罚
  - 历史背景：海外航线与商人集团寻求王室支持
  - 发生机制：第一关常规池，权重 2；第 1 年固定开场前缀之一
5. **贵族阻挠 `politicalGridlock`**
  - 费用：2
  - 标签：有害
  - 处理：支付 2 经费
  - 不处理：获得`权力流失`3 回合（每回合抽牌尝试 -1）
  - 历史背景：大贵族联合牵制新税与政令推进
  - 发生机制：第一关常规池，权重 2
6. **地方总督坐大 `powerVacuum`**
  - 费用：2 或干预
  - 标签：有害 + 持续
  - 处理：支付 2 或用干预
  - 不处理：年末登记“升级为王权危机”（非立即数值惩罚）
  - 历史背景：地方权力中心绕开宫廷形成事实自治
  - 发生机制：第一关常规池，权重 1；属于升级调度源头
7. **王权危机 `majorCrisis`**
  - 费用：仅干预
  - 标签：有害 + 持续
  - 处理：必须干预
  - 不处理：每年末合法性 -1 且下回合抽牌修正 -1，并持续留场
  - 历史背景：中央权威受挑战后演变为结构性治理危机
  - 发生机制：仅由`地方总督坐大`未处理后在下一事件阶段转化产生

#### 2) 脚本历史事件（第一关）

1. **遗产战争 `warOfDevolution`**
  - 费用：脚本“进攻”消耗 2（非普通付费解事件）
  - 标签：史实
  - 处理：可发动进攻（权力 +1；50% 概率财政 +1）
  - 不处理：无直接惩罚，但会占位到 1669 窗口结束
  - 历史背景：1667–1668 年法兰西围绕西属尼德兰/弗朗什孔泰发动战争，促发欧洲制衡
  - 发生机制：仅在 1667–1669 年脚本注入

## 第 2 关：烈日长影（`secondMandate`，1676–1700）

- 回合：25 年（每回合 1 年）
- 开局资源：
  - 关卡定义值：财政 3 / 权力 3 / 合法性 3
  - 实际直开（standalone 草案默认）：财政 7 / 权力 7 / 合法性 5
  - 承接模式（continuity）：继承第一关结算资源（并按分支追加修正）
- 胜利条件（以运行时代码为准）：
  - 当前年需 ≥ 1696
  - `欧洲警觉`必须已解除（通常通过处理`里斯维克和约`）
  - 玩家身上不能挂有`遏制胡格诺-剩余势力`（`huguenotContainment`）状态；只要镇压链未完成（即`剩余势力`未归零），即便其他条件满足也不会胜利
  - 合法性 ≥ **6**（`evaluateVictory` 使用 `victoryRule.minLegitimacy`，当前值见 `src/levels/sunking/chapters/secondMandate.ts`）
  - 注：第二关 `winTargets`（财政 10 / 权力 8 / 合法性 10）**仅作界面/数据占位**，**不参与**门禁型判胜；其中「合法性 10」与通关所需的 **合法性 ≥ 6** 不是同一道门槛。`evaluateVictory` 在以上四条全部满足时判胜。

### 一、机制列表（按“发生顺序/优先级”）

1. **欧洲警觉（开章+年度）**
  - 开章即时权力损失：当前版本为 **0**（已取消开章扣权机制）。
  - 年度基础漂移（回合开始）：
    - 设 `x = 财政 + 权力 + 合法性`，`y = 欧洲警觉进度`，`k = x - 12 - y*3`。
    - 若 `k > 0`：有 `k*10%` 概率进度 `+1`。
    - 若 `k < 0`：有 `(-k)*10%` 概率进度 `-1`。
    - 概率按 100% 封顶，进度始终夹在 `1~10`。
  - 年度外压注入（事件阶段）：
    - 进度 `1~5`：以 `进度*20%` 概率额外注入 1 个欧洲警觉事件。
    - 进度 `6~10`：保底注入 1 个，再以 `(进度-5)*20%` 概率注入第 2 个。
  - 阶段文本（状态栏背景）：
    - `1~2` 缓和，`3~4` 警觉，`5~6` 遏制，`7~8` 敌视，`9~10` 冲突。
2. **脚本历史线注入窗口**
  - 按年份窗口注入：`扩张余震/谨慎王权` → `奈梅亨和约` → `撤销南特敕令` → `奥格斯堡同盟` → `九年战争` → `里斯维克和约`。
  - 注：`地方战事`不属于脚本注入，而是欧洲警觉补充池中的额外外压事件。
  - **直开第二关（standalone）的第 1 年特例**：A 槽固定 `凡尔赛开支 versaillesExpenditure`，B 槽固定 `税收反弹 taxResistance`；这一回合跳过常规的程序事件填充，因此开场固定只有这 2 个事件。承接（continuity）模式下不触发此特例。
3. **反法情绪阈值机制**
  - 当 `权力 + 财政 > 20` 时激活`反法情绪`状态。
  - 当总和刚超过 20 时即立刻 `+1` 费用；之后每再满额外 `+5`，再 `+1`（即 `ceil((权力+财政-20)/5)`）。
  - 当该状态存在时，每回合结束向抽牌堆加入 1 张`反法遏制`（抽到时以 50%/50% 概率使权力 -1 或合法性 -1；打出费用为 `max(1, floor(欧洲警觉进度/2))`，与 `src/logic/cardCost.ts` 一致，进度很低时仍至少为 1；打出后直接消失，不进入弃牌堆）。
  - 当降回阈值内时自动移除该状态并取消上述费用加成。
4. **宗教宽容的副作用触发**
  - 存在`宗教宽容`时，每回合事件相位末进行一次掷骰，按互斥三段 15% / 15% / 15%（合计 45%）至多注入三类宗派事件之一：`詹森派争议` / `阿米尼乌斯派争论` / `胡格诺地下网络骚动`；若三者已有任意一个占用槽位，或当前没有空槽，则跳过本次注入。
5. **胡格诺镇压链机制**
  - 选择“严厉镇压”后：添加`遏制胡格诺-剩余势力(3)`并向牌库加入 3 张`镇压胡格诺派`。
  - 每打出 1 张，剩余势力 -1；归零时移除该状态并清除所有该临时牌。
  - **胡格诺残余卷土重来**：只要`遏制胡格诺-剩余势力`仍在场，每**两次** `runEventPhase`（即正常情况下每两个回合一次，触发点在事件相位末）会向抽牌堆插入 1 张`镇压胡格诺派`，同时使`遏制胡格诺-剩余势力`+1，模拟地下网络的反复重建；状态被清零时该机制随之失效。
  - 注：`镇压胡格诺派` 在代码中实际标签为 `extra`（不是 `temp`）；其“打出后不入弃牌堆、不进入弃牌循环”的行为来自 `gameReducer.PLAY_CARD` 内的专用分支，而非 `temp` 标签。
  - **严格不变量**：`遏制胡格诺-剩余势力`的数字始终等于玩家整个牌库（抽牌堆 + 手牌 + 弃牌堆）中`镇压胡格诺派`卡牌的实例总数。每次新增/打出该卡或读档（`HYDRATE`/`normalizeGameState`）时都会即时通过 `enforceHuguenotContainmentInvariant` 同步：若卡数下降至 0，状态被移除并重置“卷土重来”计数器；若数字与实卡数不一致（旧存档/外部 HYDRATE），状态会被强制改写为实际卡数；游离在 `cardsById` 中但已不在任何区域的实例会被一并清理。

#### 2) 第三关程序池 · 王位线四事件（+ `successionTrack` / 对手摸牌）

> **门控**与铺张方式见「一、5」：`successionTrack === 0` 时四者权重为 0，队列中残留队首在抽牌时会被**丢弃**直至进度离开 0。  
> **「倒戈 / 选侯」类处理代价**：`onFundSolveEffects` 中含 **`opponentNextTurnDrawModifier: +1`** 时，表示下一年 **哈布斯堡** 年初多摸 1 张（与玩家「大同盟渗透外交」的减免是不同系统）。

1. **倒戈试探（Bavarian defection probe）** `bavarianCourtRealignment`：解付 **2** → 进度 **+1**；**且对手下年初摸牌 +1**（`opponentNextTurnDrawModifier`）。年末未处理 **−1** 进度。权重 **2**。（历史：巴伐利亚选侯与波旁/大同盟/帝国站队。）

### 四、状态与关卡态字段（效果 / 发生原因）

> 下列为**关卡态字段**（非 `playerStatuses` 条目）；名称附英文便于对照代码与 UI。

1. **`successionTrack`：王位争夺进度（Succession track，−10～+10）**
  - 开局 **0**；危机与事件/对手效果增减；**±10** 立即终局（胜 / `defeatSuccession`）。区间收官见「关卡元信息」。
2. **`opponentHabsburgUnlocked`：哈布斯堡对手已解锁（Habsburg opponent unlocked）**
  - `false` → `true`：`completeSuccessionCrisisAndRevealOpponent`（危机处理完毕或年末未处理链条末尾）。
3. **`opponentStrength`：对手实力 / 对手费用上限（Opponent strength / budget）**
  - 初值 **2**；`dualFrontCrisis` 的主动选择或年末未处理惩罚会使其 **+1**。
4. **`opponentCostDiscountThisTurn`：本回合对手费用减免（Opponent cost discount this turn）**
  - 打出 **大同盟渗透外交（Grand Alliance infiltration diplomacy）** `grandAllianceInfiltrationDiplomacy` 时置 **1**；**下一年** `opponentBeginYearDrawPhase` **清零**后再摸牌。
5. **`opponentNextTurnDrawModifier`：对手下年初摸牌修正（Opponent next-year draw modifier）**
  - 由第三关程序事件解付效果累加（如 `bavarianCourtRealignment`、`imperialElectorsMood` 的 `opponentNextTurnDrawModifier: +1`）；**下一年** `opponentBeginYearDrawPhase` 计入 `max(0, 1 + …)` 摸牌后**归零**。
6. **`utrechtTreatyCountdown`：乌得勒支和约倒计时（Utrecht treaty countdown）**
  - **1713** 首次注入和约为 **6**；和约事件仍占场且未主动 **结束战争** 时，**每年末** −1；到 **0** 与主动「结束战争」相同，触发 `stateAfterUtrechtTreatyEndsWar`。
7. **`warEnded`：战争叙事已结束（War narrative ended）**
  - 玩家主动结束或倒计时归零；**关闭**哈布斯堡对手阶段（见上）。**`utrechtSettlementTier`**：仅在此时写入，用于胜利overlay的条约结语分支。
8. **`utrechtSettlementTier`：和约时的进度档位（Utrecht settlement tier）**
  - 取值 `bourbon` / `compromise` / `habsburg`，由签约瞬间的 `successionTrack` 映射（**≥+5** / **−4～+4** / **其余**）；最终胜利且该字段非空时，UI 显示 `outcome.utrechtVictoryEpilogue.*`，不再重复显示无和约路径的 `outcome.successionTier.*`。
9. **`successionOutcomeTier`：收官结局档位（Succession outcome tier）**
  - 仅在**未**走和约结语分支、**关卡收官**且 **−10 < 序 < +10** 时由 `successionIntervalTier` 写入（波旁 / 妥协 / 哈布斯堡三档）；若已签和约（`utrechtSettlementTier` 非空），胜利 overlay 以和约结语为准。

### 五、叙事参考（导语 / 文案基底）

西班牙哈布斯堡末王卡洛斯二世无嗣，帝国遗产危及欧洲均势；波旁以安茹公爵腓力（费利佩五世）系血缘与遗嘱自辩，哈布斯堡以奥地利大公查理系延续自辩；战争升级为大同盟式多国冲突。1713《乌得勒支和约》等确立费利佩可留西班牙王位但法西不得合并，条约均势取代单一普世王朝帝国想象。路易十四留下的强国家机器与欧洲遏制记忆，在本关可作为“高能力 / 高外部反弹”的双刃叙事。

- **可收束性**：**乌得勒支和约（Treaty of Utrecht）** 与 **1720 收官**、区间结局已写入逻辑；若测试仍混淆「点和约是否立即终局」，再补 UI 提示文案即可。
